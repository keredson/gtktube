# GTKTube

GTKTube is a local privacy-first Python/GTK4 YouTube player.
It stores subscriptions and viewing history locally in SQLite, and does not use a Google account.
The feed has no algorithm.  It just shows recent videos from the channels you subscribe to.

## Install

```sh
pip install gtktube
```

## Run
This is a desktop app, so you can just use your normal linux app launcher.  Or from command line:

```sh
gtktube
```
